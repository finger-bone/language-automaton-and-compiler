Language

Symbol​

A symbol is an atomic unit of the language. We usually just use characters starting from aaa or use numbers.

Sentence​

A sentence is a sequence of symbols. We usually use s,u,v,w,x,y,zs, u, v, w, x, y, zs,u,v,w,x,y,z as notation for a sentence. That is, for example, u=aaaau=aaaau=aaaa.

Language​

Language is a set of sentences. Usually noted as L\mathcal{L}L.

At Operator​

We can use n@wn@wn@w to take the nnn th symbol of sentence www, starting from zero. For example,

0@(abc)=a0@(abc) = a0@(abc)=a

Vocabulary​

A vocabulary is a set of symbols. Usually noted as V\mathcal{V}V.

Length of the Sentence​

The length of a sentence is how many symbols it has. Noted as ∣v∣|v|∣v∣ for sentence vvv.

Empty Sentence​

An empty sentence is a sentence with no words. It is unique and often noted as λ\lambdaλ, ∣λ∣=0|\lambda| = 0∣λ∣=0.

Concatenation​

The concatenation of two sentences uuu and vvv is denoted as uvuvuv.

Prefix and Suffix​

sss is a suffix of yyy if and only if there exists such xxx that, xs=yxs=yxs=y.

sss is a prefix of yyy if and only if there exists such xxx that, sx=ysx=ysx=y.

Substring​

sss is a substring of yyy if and only if there exists such xxx and zzz that, xsz=yxsz=yxsz=y.

Sentence with Single Symbol​

If a sentence has only one symbol, we can use the symbol to denote the sentence. For example, aaa can be a sentence with one symbol aaa.

Repeat​

If a sentence uuu is nnn times concatenation to vvv, then we note, u=vnu = v^nu=vn.

We define, u0=λu^0 = \lambdau0=λ.

Concatenation with Vocabulary​

If we concatenate a sentence www to a vocabulary V\mathcal{V}V, it yields a language set.

wV={ws∣s∈V}w\mathcal{V} = \{ ws | s \in \mathcal{V} \}wV={ws∣s∈V}

Language Concatenation​

The concatenation of two language L1\mathcal{L}_1L1​ and L2\mathcal{L}_2L2​ yields a new language, such that every sentence is a concatenation of a sentence in L1\mathcal{L}_1L1​ and a sentence in L2\mathcal{L}_2L2​.

Similarly, if the concatenation happens multiple times, we note it as exponentiation.

Vocabulary as Language​

We define,

λV\lambda \mathcal{V}λV

As the corresponding language set of a vocabulary. We still note this language V\mathcal{V}V.

Closure​

The star closure of a language results in all the possible sentence that can be built by concatenation.

That is,

For positive closure, it removes the empty sentence.
