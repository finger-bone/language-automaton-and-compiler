Pumping Lemma of Regular Languages

The pumping lemma for regular languages is a lemma that describes an essential property of all regular languages. Informally, it says that all sufficiently long strings in a regular language may be pumped—that is, have a middle section of the string repeated an arbitrary number of times—to produce a new string that is also part of the language.

In the following passage, we only talk about infinite languages. For finite language is guaranteed to be regular language.

Pumping Lemma of Regular Languages and Its Proof​

Pumping Lemma states that, if L\mathcal{L}L is an infinite regular language, there must exists a pumping length p≥1p \geq 1p≥1 (ppp is only dependent on the language).

If an integer ppp is a pumping length. For any w∈Lw \in \mathcal{L}w∈L and ∣w∣≥p|w| \geq p∣w∣≥p, there must exists a way of dividing www into,

w=xyzw = xyzw=xyz

Where, ∣y∣≥1|y| \geq 1∣y∣≥1 and ∣xy∣≤p|xy| \leq p∣xy∣≤p.

And ∀n≥0\forall n \geq 0∀n≥0, xynz∈Lxy^nz \in \mathcal{L}xynz∈L.

This can be proven with regular expression- and is very simple.

Because L\mathcal{L}L is infinite, there must exist at least one star, that is,

R=r1r2∗r3R = r_1r_2^*r_3R=r1​r2∗​r3​

Where r2r_2r2​ is not empty.

Thus, if xxx matches r1r_1r1​, zzz matches r3r_3r3​, yyy matches r2r_2r2​, we will have the pumping lemma.

Example​

We usually use pumping lemma to prove a language is not regular.

For example, we would like to prove,

0n1n0^n1^n0n1n

Is not a regular language.

We assume there exists a pumping length ppp, and assume that this is a regular language.

Pumping lemma tells us that for every string longer than pumping length, there must exists a pumping decomposition. We can take,

w=0p1pw = 0^p1^pw=0p1p

We know that a pumping decomposition exists, but we don't know exactly how- it's okay, we can assume,

w=xyzw = xyzw=xyz

No matter how www decomposes, there are only there cases for yyy.

  • yyy contains only 111, then obviously, xy2z∉Lxy^2z \not \in \mathcal{L}xy2z∈L, because there will be more 111 than 000.
  • yyy contains only 000, then obviously, xy2z∉Lxy^2z \not \in \mathcal{L}xy2z∈L, because there will be more 000 than 111.
  • yyy contains 000 and 111, then y=0a1by = 0^{a}1^{b}y=0a1b, then xy2z=0n1b0a1n∉Lxy^2z=0^{n}1^b0^a1^{n} \not \in \mathcal{L}xy2z=0n1b0a1n∈L

Thus, there doesn't exists such a decomposition.

In conclusion, 0n1n0^n1^n0n1n is not a regular language.
