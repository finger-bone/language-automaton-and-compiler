Regular Expression

Regular expression is a simpler way to describe a regular language.

Regular Expression​

Definition​

Regular expression is an expression that can yield a language (later proved to be a regular language). We note L(⋅)\mathcal{L}(\cdot)L(⋅) as the language generated by the regular expression ⋅\cdot⋅.

Regular expression is defined as,

Any sentence vvv is a valid regular expression, L(v)={v}\mathcal{L}(v) = \{ v \}L(v)={v}.

The ∗*∗ of a regular expression rrr is still a regular expression, L(r∗)=L(r)∗\mathcal{L}(r^*) = \mathcal{L}(r)^*L(r∗)=L(r)∗

The +++ of a regular expression rrr is still a regular expression, L(r+)=L(r)+\mathcal{L}(r^+) = \mathcal{L}(r)^+L(r+)=L(r)+

If rrr is a regular expression, rnr^nrn is also a regular expression, L(rn)=L(r)n\mathcal{L}(r^n) = \mathcal{L}(r)^nL(rn)=L(r)n

If r1r_1r1​ and r2r_2r2​ are regular expressions, then r1r2r_1 r_2r1​r2​ is also a regular expression, L(r1r2)=L(r1)L(r2)\mathcal{L}(r_1r_2) = \mathcal{L}(r_1)\mathcal{L}(r_2)L(r1​r2​)=L(r1​)L(r2​).

If r1r_1r1​ and r2r_2r2​ are regular expressions, then r1+r2r_1 + r_2r1​+r2​ is also a regular expression, L(r1+r2)=L(r1)∪L(r2)\mathcal{L}(r_1 + r_2) = \mathcal{L}(r_1) \cup \mathcal{L}(r_2)L(r1​+r2​)=L(r1​)∪L(r2​)
